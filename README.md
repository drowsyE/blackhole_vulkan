## Simple black hole simulation with vulkan backend

### Camera control
w - move forward
a - move left
s - move backward
d - move down

q - move down
e - move up

up arrow - rotate up
left arrow - rotate left
down arrow - rotate down
right arrow - rotate right

Window is resizable. Be aware that increasing the size of the window will increase the amount of computation.
